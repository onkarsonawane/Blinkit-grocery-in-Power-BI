📊 Project Overview
This project analyzes Blinkit grocery sales data using Python and Power BI to uncover meaningful business insights.
The objective is to understand sales performance, product demand, outlet performance, and customer buying patterns through data analysis and visualization.
The project demonstrates how data analytics can help businesses make data-driven decisions by identifying trends and patterns in sales data.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Compare sales across outlet types and sizes
Understand customer purchasing patterns
Build an interactive Power BI dashboard

🛠️ Tools & Technologies Used
Python
Pandas
Power BI
Data Visualization
Data Cleaning & Data Analysis

📂 Project Files
Blinkit_Analysis_in_Python.ipynb → Data cleaning and preprocessing using Python
blinkit.pbix → Interactive Power BI dashboard file
dataset.csv → Blinkit sales dataset used for analysis

📈 Dashboard Features
The Power BI dashboard includes the following insights:
Total Sales Overview
Sales by Product Category
Sales by Outlet Type
Outlet Size Performance
Customer Demand Trends

These visualizations help in understanding which products and outlets generate the highest revenue.

🔍 Key Insights
Certain product categories generate significantly higher sales.
Larger outlet sizes tend to contribute more to overall revenue.
Sales trends vary depending on outlet type and product category.
Data visualization makes it easier to identify business opportunities.

🚀 How to Use
Download the repository.
Open the .pbix file using Power BI Desktop.
Explore the interactive dashboard and visualizations.
